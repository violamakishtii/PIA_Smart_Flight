## 🧠 Project Overview

This project is a complete **end-to-end Machine Learning and Neural Network pipeline** built using the *PIA 2026 Smart Flight Dataset*.  

It focuses on predicting whether a flight will be:

- ✔ On Time  
- ❌ Delayed  

The project also explores deeper insights such as:
- Airline operational efficiency
- Passenger behavior patterns
- Weather impact on delays
- Revenue and fuel consumption relationships

This is not just a simple ML model — it is a **real-world aviation intelligence system simulation**.

## 📂 Dataset Description

The dataset contains 800 flight records with the following features:

- Flight duration
- Departure & arrival cities
- Aircraft type
- Passenger count
- Seat capacity
- Load factor
- Ticket price
- Revenue
- Delay minutes
- Weather conditions
- Fuel consumption
- CO₂ emissions
- Customer ratings
